# Blog

The skeleton ships with a basic multilingual blog template. You can access the blog content via Laravel Nova.

# Remove Blog component

If you don't need any of these components, feel free to remove the following directories

  • app/Domain/Blog
  • database/factories/Blog
  • resources/vies/app/pages/blog

... and the following files ...

  • database/migrations/2022_03_08_111314_create_blog-posts_table.php
  • database/migrations/2022_03_08_111441_create_blog-categories_table.php
  • database/migrations/2022_03_08_111523_create_blog-post_categories_table.php
  • database/migrations/2022_03_08_111548_create_blog-post_localized_content_table.php
  • database/seeders/BlogSeeder.php

Further, remove any references to the Blogs in ...

  • app/App/Providers/NovaServiceProvider.php
  • app/Support/Enums/TableName.php
  • database/seeders/DatabaseSeeder.php
  • routes/web.php
  • config/filesystems.php
